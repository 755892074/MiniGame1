## 抖音小游戏出包环境搭建

### 环境配置

- Colorspace Linear → Gamma（WebGL 兼容）

### 问题修复
1. **Editors → Editor 重命名**：Unity 只认 `Editor` 文件夹名，不认 `Editors`，导致 WebGL 编译时报错
2. **MiniGame1Build.cs 编译错误**：缺 `UnityEditor.Build.Reporting` using
3. **颜色空间不兼容**：WebGL 不支持 Linear，改为 Gamma
4. **旧格式打包 brotli 压缩**：需取消勾选，否则 game.js 被压缩导致抖音编译失败

### 字体修复
