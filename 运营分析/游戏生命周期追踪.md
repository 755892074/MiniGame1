# 铲屎官疯了 v2 — 游戏生命周期与运营追踪

> 文件版本：v0.1 | 创建：2026-07-20 | 维护：持续跟进
> 目的：把游戏循环、成长路线、经济闭环、留存目标、版本数据跟进集中到一处，方便上线后分析整个游戏生命周期（LTV / 留存 / 变现）。
> 说明：本表为「运营北极星追踪」，数值目标取自 GDD / 成长系统设计 / IAA 数值参考；**真实数据在游戏上线后回填**。

## 一、游戏核心循环（Core Loop）

| 层级 | 循环内容 | 触发频率 | 备注 |
|------|----------|----------|------|
| 微观循环 | 取食物栈顶 → 喂队首宠物 → 满碗宠物离开 → 下一只上前 | 每关多次 | 栈顶+队列+中转碗，算法保证可解 |
| 关卡循环 | 选关 → 解谜 → 结算（星评+奖励+看广告翻倍）→ 下一关 | 每局 1 关 | 50 关软启动 |
| 日常循环 | 每日签到 → 领小鱼干/徽章 → 推进关卡/住所升级 | 每日 | 连续 7 天签到，第 7 天给彩虹毛球 |
| 成长循环 | 赚经验升级铲屎官 → 解锁新宠物/住所外观 → 收集成就 | 跨日 | 等级=打工线，称号带来成就感 |

**核心付费钩子**：连胜倍数奖励累加 → 死局时花 900 金币（或 1 次激励视频）保连胜，形成强烈续命动机。

## 二、成长路线（Progression）

| 系统 | 资源 | 成长方式 | 解锁内容 |
|------|------|----------|----------|
| 铲屎官等级（打工线） | 打工经验 exp | 通关积累（每星 30 exp） | 称号变化、解锁新宠物/外观 |
| 住所升级 | 小鱼干 | 花小鱼干升级 铁笼→纸箱→… | 外观变化 |
| 宠物入住 | 小鱼干 / 喂食 N 次 | 按等级解锁，每只 3 阶段 | 6 只宠物全入住 |
| 救助徽章 | 3 星通关 | 每 3 星 1 枚 | 特殊区域/外观（需徽章） |
| 成就收集 | — | 各类行为达成 | 留存辅助 |

> 全部 50 关 3 星 ≈ 4000 小鱼干（不含翻倍），足够前期建设。
