# 01 — 宠物帧动画 AI 提示词

> 6只宠物 × 5个动作 × 3个成长阶段。共需约 240 帧。
> 生成顺序：先出设定图（锚点）→ 再用图生图出动画帧。

---

## 使用方法

1. 先用「设定图」提示词生成 1 张参考图
2. 把参考图作为 image prompt 输入，再用「帧动画」提示词生成每帧
3. 所有帧保存到 `_raw/{pet}_{action}_stage{N}/` 目录
4. 运行 `scripts/batch_process.py` 自动切图重命名

---

## 全局风格后缀（每条提示词末尾都加）

```
chibi style, flat design, thick black outlines, pastel color palette, solid white background, centered composition, no shadow, 128x128
```

## 帧动画生成（基于设定图用 image-to-image）

### 动作模板

把 `{CHARACTER}` 替换为对应宠物描述，`{N}` 替换为帧序号。

#### idle（待机呼吸，4帧）
```
{CHARACTER}, idle breathing animation, frame {N} of 4,
frame 1: normal relaxed pose
frame 2: body slightly squished down (breathing in)
frame 3: body more squished (full breath)
frame 4: back to normal relaxed pose,
same character design, same colors, same style,
chibi style, flat design, thick outlines, white background, 128x128
```

| 帧 | 小橘 | 阿黄 | 球球 | 翠翠 | 泡泡 | 麻薯 |
|---|---|---|---|---|---|---|
| 1 | 正坐 | 正坐 | 正坐 | 栖息 | 悬停 | 正坐 |
| 2 | 微缩 | 微缩 | 腮帮鼓 | 翅膀收 | 尾巴摆 | 耳朵抖 |
| 3 | 更缩 | 更缩 | 腮帮更鼓 | 翅膀张 | 尾巴大幅 | 耳朵大幅 |
| 4 | 回正 | 回正 | 回正 | 回正 | 回正 | 回正 |

#### happy（开心，4帧）
```
{CHARACTER}, happy celebration animation, frame {N} of 4,
frame 1: eyes closed in smile, body normal
frame 2: jumping up slightly, arms/paws up, big smile
frame 3: at peak of jump, sparkles around, biggest smile
frame 4: landing back, still smiling,
same character design, same colors,
chibi style, flat design, thick outlines, white background, 128x128
```

#### cry（哭泣害怕，4帧）
```
{CHARACTER}, sad crying afraid animation, frame {N} of 4,
frame 1: head down, eyes watery
frame 2: tears falling, body trembling
frame 3: more tears, hugging self
frame 4: curled up slightly, still crying,
same character design, same colors,
chibi style, flat design, thick outlines, white background, 128x128

#### run（跑动，6帧）

#### cute（眨眼卖萌，2帧）

frame 1: eyes open, slight smile

same character design, same colors,
chibi style, flat design, thick outlines, white background, 128x128
```

## 三阶段外观变体

每只宠物的每个动作，都要生成 3 个版本：

### 阶段1：脏兮兮（刚救助）
在角色描述前加：
```
dirty, wet messy fur, sad tired expression, greyish dull tint, 
thinner body posture, slightly shivering, 
```

### 阶段2：干净（基本恢复）= 默认版
使用设定图原本的描述，不加修改。

### 阶段3：幸福（完全康复）
在角色描述前加：
```
glowing healthy fur, bright vivid colors, extra fluffy and round body, 
happy sparkling eyes, small sparkle effects around, 
```

## 输出文件命名

生成后把帧图片放到对应目录，命名如下：

```
_raw/
├── cat_idle_stage1/          ← 小橘 idle 阶段1
│   ├── frame_01.png
│   ├── frame_02.png
│   ├── frame_03.png
│   └── frame_04.png
├── cat_idle_stage2/          ← 小橘 idle 阶段2
├── cat_idle_stage3/
├── cat_happy_stage2/
├── dog_idle_stage2/
└── ...
```

脚本会自动处理：
- 去白底 → 裁切到 128×128 → 重命名 → 部署到 `Assets/Resources/ArtPets/{pet}/{action}_s{N}/frame_XX.png`
