## 输出命名与部署

```
_raw/
├── yard_bg_lv1.png ~ yard_bg_lv5.png    ← 小院背景
├── chapter_01.png ~ chapter_05.png      ← 章节封面
```

部署路径：
- 背景 → `Assets/Resources/ArtScenes/yard_bg_lv{N}.png`
- 封面 → `Assets/Resources/ArtChapters/chapter_0{N}.png`
