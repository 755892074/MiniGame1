> 音效不能 AI 生成，需要从素材库获取或自制。

## 音频格式规范

| 属性 | 规范 |
|---|---|
| 格式 | WAV（Unity原生）或 OGG（体积小） |
| 采样率 | 44100 Hz |
| 声道 | 单声道（音效）/ 立体声（BGM） |
| 音量 | 标准化到 -3dB |
| 文件大小 | 音效 < 100KB，BGM < 500KB |

---

## 部署路径

```
Assets/Audio/
├── SFX/        ← 音效 (S01~S13)
│   ├── pick_food.wav
│   ├── pour_food.wav
│   └── ...
└── BGM/        ← 背景音乐 (S14~S15)
    ├── bgm_main.wav
    └── bgm_game.wav
```
