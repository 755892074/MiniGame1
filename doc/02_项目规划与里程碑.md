## 4. 广告变现策略

### 4.1 广告位详细设计

**找茬大师**：

| 广告位 | 位置 | 触发 | 频控 |
|--------|------|------|------|
| 激励视频-翻倍 | 过关结算弹窗 | 用户主动点击 | 每关1次 |
| 激励视频-提示 | 游戏中提示按钮 | 用户主动点击 | 每关1次 |
| 插屏-关间 | 过关→下一关 | 系统触发 | 每3关1次 |
| Banner | 主界面底部 | 常驻 | 无 |

**无厘头大挑战**：

| 广告位 | 位置 | 触发 | 频控 |
|--------|------|------|------|
| 激励视频-复活 | 生命耗尽弹窗 | 用户主动点击 | 每局1次 |
| 激励视频-道具 | 道具不足弹窗 | 用户主动点击 | 每局3次 |
| 插屏-关间 | 连续答对10题后 | 系统触发 | 每10题1次 |
| Banner | 主界面底部 | 常驻 | 无 |

### 4.2 广告频控策略

- 单次游戏 session 激励视频最多 5 次
- 插屏广告最小间隔 30 秒
- 新用户前 3 关不展示插屏（保护体验）
- 过关后结算界面优先展示翻倍按钮，不强制弹广告

## 5. 数据埋点方案

### 5.1 关键事件

| 事件 | 参数 | 用途 |
|------|------|------|
| game_start | 关卡ID | 关卡启动率 |
| level_complete | 关卡ID, 用时, 得分, 是否看广告 | 关卡通过率 |
| level_fail | 关卡ID, 用时, 进展 | 失败分析 |
| ad_impression | 广告类型, 位置 | 广告展示量 |
| ad_click | 广告类型, 位置 | 广告点击率 |
| ad_reward | 广告类型 | 激励完成率 |
| session_start | 用户来源 | DAU |
| session_end | 时长 | 平均游戏时长 |
| share | 分享类型 | 裂变效果 |

### 5.2 核心看板指标

- DAU / MAU
- 次日留存 / 7日留存 / 30日留存
- 平均游戏时长
- 关卡通过率（分关卡）
- ARPU / ARPPU
- 广告填充率 / eCPM
- 分享率 / 裂变系数
