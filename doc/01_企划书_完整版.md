# 01 企划书 - 抖音2D小游戏双产品线

---

## 1. 项目概述

### 1.1 项目定位

在抖音小游戏平台推出两款 IAA 变现的 2D 休闲小游戏，通过 AI 工具链（WorkBuddy + Image2 + SeedDance）实现全流程 AI 驱动开发，验证"单人 + AI 即可完成商业小游戏开发"的可行性。

### 1.2 产品矩阵

| | 找茬大师 | 无厘头大挑战 |
|---|---|---|
| **类型** | 找茬（Spot the Difference） | 问答（Absurd Quiz） |
| **目标用户** | 18-40 岁女性为主，休闲玩家 | 18-35 岁泛人群，搞笑内容消费者 |
| **单局时长** | 60-120 秒 | 10-30 秒/题 |
| **爽感来源** | 找出差异的成就感 | 答对荒诞题的意外感 |
| **传播机制** | 挑战好友分数 | 截图分享搞笑题目 |
| **变现模式** | 激励视频 + 插屏 | 激励视频复活 + 插屏 |

## 2. 游戏1：找茬大师

### 2.1 核心玩法

```
┌──────────────────────────────┐
│         找茬大师             │
│                              │
│  ┌──────────┐ ┌──────────┐  │
│  │  原图 A  │ │  对比图 B │  │
│  │          │ │          │  │
│  │  [差异1] │ │  [差异1] │  │
│  │  [差异2] │ │  [差异2] │  │
│  │  [差异3] │ │  [差异3] │  │
│  │   ...    │ │   ...    │  │
│  └──────────┘ └──────────┘  │
│                              │
│  ⏱ 45s  🔍 剩余5处  ⭐ 120分│
└──────────────────────────────┘
```

### 2.2 关卡结构

每关包含：
- **原图**：AI 生成的场景图（Image2）
- **对比图**：同一场景微调后的图（Image2 同 prompt + 变化指令）
- **差异点列表**：坐标 + 半径 + 描述（JSON 数据）
- **时间限制**：根据难度 30-90 秒
- **差异数量**：3-8 处

### 2.3 难度曲线

| 章节 | 关卡数 | 差异数 | 时限 | 图片复杂度 | 差异大小 |
|------|--------|--------|------|------------|----------|
| 新手村 | 1-10 | 3 | 60s | 简单场景 | 明显 |
| 小镇 | 11-30 | 4-5 | 50s | 中等场景 | 中等 |
| 大都市 | 31-50 | 5-6 | 45s | 复杂场景 | 细微 |
| 奇幻世界 | 51-70 | 6-7 | 40s | 繁复场景 | 隐蔽 |
| 大师殿堂 | 71-100 | 7-8 | 30s | 极繁复 | 极隐蔽 |

### 2.4 用户体验流程

```
启动 → 主界面 → 选关 → 加载关卡 → 计时开始
       ↓
   [找茬中] → 点到正确位置 ✅ +得分+特效
       ↓     → 点到错误位置 ❌ -时间惩罚
   [全部找出] → 过关结算 → 激励视频(翻倍) → 下一关
   [时间耗尽] → 失败 → 重试/插屏广告 → 返回主界面
```

## 3. 游戏2：无厘头大挑战

### 3.1 核心玩法

```
┌──────────────────────────────┐
│    无厘头大挑战    🔥连击×8  │
│                              │
│   什么东西越洗越脏？         │
│                              │
│  ┌────────────────────────┐  │
│  │  A. 水           ✅    │  │
│  ├────────────────────────┤  │
│  │  B. 毛巾          ❌    │  │
│  ├────────────────────────┤  │
│  │  C. 手帕          ❌    │  │
│  ├────────────────────────┤  │
│  │  D. 洗洁精        ❌    │  │
│  └────────────────────────┘  │
│                              │
│  ⭐ 2400分  ❤️❤️❤️  第24题  │
└──────────────────────────────┘
```

### 3.2 题库设计

| 分类 | 示例 | 占比 |
|------|------|------|
| 脑筋急转弯 | "什么东西越洗越脏？→ 水" | 30% |
| 反常识冷知识 | "地球上最多的生物是？→ 细菌" | 25% |
| 视觉错觉题 | 配图 + "图中有几个三角形？" | 20% |
| 文字游戏 | "什么门永远关不上？→ 球门" | 15% |
| 抖音热梗 | 基于当下热门梗的题目 | 10% |

### 3.3 游戏机制

- **连击系统**：连续答对叠加倍率（×1 → ×2 → ×3 → ... → ×MAX）
- **生命值**：3 格生命，答错扣 1 格
- **复活机制**：生命耗尽 → 看激励视频 → 满血复活 + 保留连击
- **道具系统**：
  - 🔍 排除两个错误答案（激励视频获取 ×1）
  - ⏱ 加时 10 秒
  - 💡 提示（激励视频获取 ×1）

### 3.4 社交裂变

- **挑战模式**：生成专属题卡（"我连续答对 15 题，你能超过我吗？"）
- **搞笑分享**：分享单题的荒诞选项截图到抖音
- **排行榜**：好友分数 PK、全国排名

## 6. 竞品分析

### 6.1 找茬类竞品

| 竞品 | 优势 | 劣势 | 差异化 |
|------|------|------|--------|
| 全民来找茬 | 用户基数大 | UI老旧，关卡重复 | AI 生图关卡无限量+画面精美 |
| 找茬大师 | 品牌认知 | 广告过多 | 克制广告频次+体验优先 |
| 眼力大挑战 | 抖音原生 | 内容单薄 | 章节制+主题系列+收藏系统 |

### 6.2 问答类竞品

| 竞品 | 优势 | 劣势 | 差异化 |
|------|------|------|--------|
| 头脑王者 | 题库丰富 | 太严肃，不够搞笑 | 无厘头荒诞路线+视觉化题目 |
| 猜歌达人 | 垂直领域 | 受众窄 | 泛娱乐+多题型混合 |
| 成语大挑战 | 文化牌 | 年轻用户不爱 | 轻松搞笑+反套路出题 |

## 7. 风险与对策

| 风险 | 等级 | 对策 |
|------|------|------|
| 抖音审核不通过 | 中 | 严格对照审核规范，预留修改时间 |

| 竞品抄袭 | 低 | 快速迭代，AI 生成内容壁垒 |
| 苹果/安卓兼容性 | 低 | 按抖音规范打包测试 |
