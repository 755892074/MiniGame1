## 2. Image2 生图工作流

```
WorkBuddy 提供提示词 → 你在 Image2 网页版粘贴 → 生成 → 下载 → 导入项目
```

### 2.1 导入规范

1. 下载 PNG 到 `Art/Sprites/FindDiff/ChapterX/` 或 `Art/Sprites/AbsurdQuiz/`
2. 命名规则：
   - 找茬原图：`L{关卡号}_A.png`（如 `L01_A.png`）
   - 找茬对比图：`L{关卡号}_B.png`（如 `L01_B.png`）
   - 插画：`illus_{描述}.png`
3. 在 Tuanjie 中设置：
   - Texture Type: Sprite (2D and UI)
   - Compression: Normal Quality
   - Max Size: 2048

## 3. 找茬关卡图提示词模板

### 3.1 新手村 - 简单场景

**主题**：儿童房、公园、海滩、厨房

```
提示词模板：
A bright, colorful cartoon-style [场景],
viewed from a fixed camera angle, [场景描述],
clean vector art style, flat colors, high contrast,
suitable for a children's find-the-difference game,
no text, no watermarks, 16:9 aspect ratio, 1080x1920

场景示例：
- "children's bedroom with toys, bed, window, and bookshelf"
- "sunny park with trees, bench, pond, and playground"
- "kitchen with table, fruits, stove, and window"
```

**差异点生成**：提示词末尾追加描述
```
"For the second image, make these subtle changes:
1. Remove the clock from the wall
2. Change the color of the curtains from blue to green
3. Add a cat sitting on the windowsill
Keep everything else identical."
```

### 3.2 小镇 - 中等场景

**主题**：街道、市场、教室、办公室

```
提示词模板：
A detailed cartoon illustration of [场景],
warm color palette, flat design with soft shadows,
many objects and details for finding differences,
1080x1920 portrait, mobile game art style

场景示例：
- "busy street market with stalls, people, fruits, and signs"
- "classroom with desks, blackboard, students, and decorations"
- "cozy coffee shop with tables, plants, menu board, and customers"
```

### 3.3 大师级 - 繁复场景

**主题**：图书馆、博物馆、游乐场、奇幻景观

```
提示词模板：
An extremely detailed "Where's Waldo" style illustration of [场景],
filled with tiny objects, patterns, and hidden details,
rich color palette, whimsical stylized art,
matching the sophistication of a premium hidden-object game,
1080x1920 portrait

场景示例：
- "fantasy library with endless bookshelves, magical artifacts, floating candles, and mythical creatures"
- "amusement park with roller coasters, ferris wheel, crowds, food stalls, and carnival games"
```
