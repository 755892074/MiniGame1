## 2. 单元测试

### 2.1 测试框架

使用 Unity Test Framework（已内置在项目中）。

测试文件位置：`Assets/Scripts/Tests/`

### 2.2 测试覆盖清单

**找茬大师核心逻辑**：

| 测试用例 | 输入 | 预期输出 |
|----------|------|----------|
| 点击命中差异点 | 坐标(0.5, 0.5)，差异点位于(0.5, 0.5)，半径0.1 | 返回 true |
| 点击偏离差异点 | 坐标(0.5, 0.5)，差异点位于(0.2, 0.2)，半径0.1 | 返回 false |
| 全差异找出 | 依次点击所有差异点 | 触发 level_complete 事件 |
| 计时器归零 | 等待时限耗尽 | 触发 level_fail 事件 |
| 计分正确 | 找对 N 个差异 + 剩余时间 T | 分数 = N*100 + T*10 |
| 连击加分 | 连续点击正确 | 分数 × 连击倍率 |

**无厘头大挑战核心逻辑**：

| 测试用例 | 输入 | 预期输出 |
|----------|------|----------|
| 选择正确答案 | 选项索引 = correctIndex | comboCount + 1 |
| 选择错误答案 | 选项索引 != correctIndex | lives - 1, comboCount = 0 |
| 连击倍率 | comboCount = 5 | multiplier = 5 |
| 复活后状态 | 执行复活 | lives = 3, comboCount 不变 |
| 题目去重 | 出 100 题 | 无重复题目 |

### 2.3 测试命名规范

```
[模块]_[方法]_[场景]_[预期结果]

示例：
DifferenceManager_HitTest_ExactCenter_ReturnsTrue
ComboSystem_Increment_AfterFiveConsecutive_ReturnsMultiplier5
```

## 3. 集成测试（Codely 驱动）

### 3.1 场景加载测试

通过 Codely `execute_csharp_script` 验证：

```csharp
// 加载所有场景，检查是否有丢失引用
var scenes = new[] { "Main.scene", "Quiz.scene", "Splash.scene" };
foreach (var sceneName in scenes) {
    EditorSceneManager.OpenScene($"Assets/Scenes/{sceneName}", 
        OpenSceneMode.Single);
    // 检查场景中的引用完整性
    var allObjects = Object.FindObjectsOfType<GameObject>();
    foreach (var obj in allObjects) {
        var components = obj.GetComponents<Component>();
        foreach (var comp in components) {
            if (comp == null) {
                Debug.LogError($"Missing component on {obj.name} in {sceneName}");
            }
        }
    }
}
```

### 3.2 UI 面板完整性

通过 Codely 遍历所有 UI 面板，检查按钮赋值、文字引用：

```csharp
// 检查所有 Canvas 下的 Button 是否有关联的 onClick
var buttons = Object.FindObjectsOfType<Button>();
foreach (var btn in buttons) {
    if (btn.onClick.GetPersistentEventCount() == 0) {
        Debug.LogWarning($"Button {btn.name} has no onClick handler");
    }
}
```

### 3.3 关卡数据完整性

```csharp
// 检查所有 LevelData SO 的图片引用
var guids = AssetDatabase.FindAssets("t:LevelData");
foreach (var guid in guids) {
    var path = AssetDatabase.GUIDToAssetPath(guid);
    var levelData = AssetDatabase.LoadAssetAtPath<LevelData>(path);
    if (levelData.imageA == null || levelData.imageB == null) {
        Debug.LogError($"LevelData at {path} missing image reference");
    }
}
```

## 4. 自动化检查流水线

通过 Codely 一键运行所有检查：

```csharp
// 完整检查流水线
void RunFullCheck() {
    CheckSceneIntegrity();
    CheckUICompleteness();
    CheckLevelDataIntegrity();
    CheckQuestionBankIntegrity();
    CheckSpriteAtlasSettings();
    Debug.Log("Full check completed.");
}
```

在 WorkBuddy 中执行：

```
从 WorkBuddy 调用 unity_script，执行 RunFullCheck 方法
→ 输出检查报告
→ 有问题则修复
```
