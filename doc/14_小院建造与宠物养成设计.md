## 一、货币体系

### 1.1 三种货币

| 货币 | 图标 | 产出途径 | 主要用途 |
|------|------|---------|---------|
| 🪙 金币 | Gold Coin | 通关奖励、广告翻倍、每日签到 | 买地、建/升级建筑、装饰 |
| 🐟 小鱼干 | Dried Fish | 通关奖励、成年宠物定时产出 | 喂宠物涨亲密度 |
| ⭐ 徽章 | Badge | 关卡三星、成就达成 | 解锁高级宠物/稀有变异（稀缺货币） |

### 1.2 产出量设定

| 产出源 | 金币 | 小鱼干 | 徽章 |
|--------|------|--------|------|
| 通关基础 | +50×星级 | +10×星级 | 三星=1徽章 |
| 广告翻倍 | ×2 | ×2 | — |
| 成年宠物产出 | — | +5/小时/只 | — |
| 每日签到 | +20~100 | — | 每7天+1 |
| 成就达成 | +200 | +50 | +1~5 |

## 二、宠物养成系统

### 2.1 获取方式

初始送1只（橘猫），其余通过通关救助获得：
- 通关特定关卡 → 救助对应宠物 → 加入小院
- 6只宠物对应6个救助关卡（第5/10/15/20/25/30关）

### 2.2 亲密度

```
喂食小鱼干 → 亲密度+10/次
互动（每日限1次） → 亲密度+20/次
```

| 亲密度等级 | 所需累计 | 效果 |
|-----------|---------|------|
| Lv1 陌生 | 0 | 默认 |
| Lv2 认识 | 50 | 解锁新表情（😊微笑） |
| Lv3 熟悉 | 150 | 解锁新动作（蹦跳） |
| Lv4 亲近 | 300 | 日产小鱼干从1→3 |
| Lv5 亲密 | 600 | 解锁稀有变异资格 |
| Lv6 挚友 | 1000 | 专属称号 ✧XX✧ |

### 2.3 稀有变异

亲密度 Lv5 + 消耗徽章×3 → 触发变异（换色+发光特效）

| 宠物 | 变异形态 | 描述 |
|------|---------|------|
| 🐱 橘猫 | ✨ 金渐层 | 毛色从橘变金，发光 |
| 🐶 柴犬 | 🌑 黑柴 | 经典黑柴配色 |
| 🐹 仓鼠 | 🌸 樱花鼠 | 粉白色+花瓣粒子 |
| 🦜 鹦鹉 | 🌈 七彩鹦鹉 | 彩虹渐变羽色 |
| 🐟 金鱼 | 👑 龙睛金鱼 | 金色+拖尾特效 |
| 🐰 垂耳兔 | ❄️ 雪兔 | 纯白+雪花粒子 |

**实现**：换色 Shader / 叠加边框光效粒子，不需要重绘原图。

### 2.4 宠物产出（放置收益）

成年宠物每小时自动产出小鱼干，离线也计算（上限 8h）。

| 亲密度 | 产出速率 | 8h 上限 |
|--------|---------|--------|
| Lv1-3 | 1/h | 8 |
| Lv4-5 | 3/h | 24 |
| Lv6 | 5/h | 40 |

### 2.5 称号系统

| 条件 | 称号 |
|------|------|
| 1只 Lv6 | "猫之友" / "狗之友" ... |
| 3只 Lv6 | "铲屎专家" |
| 6只 Lv6 | "传说铲屎官" |
| 任意稀有变异 | "✧闪光饲主✧" |
| 6只全变异 | "✧终极收藏家✧" |

## 三、小院建造系统

### 3.1 地块与解锁

小院初始 2×2 地块，每格可放一个建筑。随铲屎官等级解锁新地块：

| 铲屎官等级 | 地块数 | 解锁等级 |
|-----------|--------|---------|
| Lv1-2 | 2×2=4 | 初始 |
| Lv3-4 | 3×3=9 | 500exp |
| Lv5-6 | 4×4=16 | 2000exp |
| Lv7-8 | 5×5=25 | 5000exp |

**买地费用**：新地块 = 100×当前地块数 金币

### 3.2 四种建筑

| 建筑 | 基础效果 | 每升1级效果 | 升级消耗 |
|------|---------|-----------|---------|
| 🥣 食盆 | 所有宠物亲密度增益 +0% | +5% | 金币递增 |
| 🧸 玩具 | 小鱼干产出速率 +0% | +10% | 金币递增 |
| 🏥 医疗 | 宠物离线产出上限 +0h | +2h | 金币+徽章 |
| 🌻 花园 | 所有宠物亲密度每日自然+0 | +2/天 | 金币+小鱼干 |

### 3.3 建筑升级消耗表

| 等级 | 食盆(金币) | 玩具(金币) | 医疗(金币+徽章) | 花园(金币+鱼干) |
|------|-----------|-----------|----------------|----------------|
| 1→2 | 100 | 100 | 150+1 | 150+50 |
| 2→3 | 300 | 300 | 400+2 | 400+120 |
| 3→4 | 800 | 800 | 1000+3 | 1000+300 |
| 4→5 | 2000 | 2000 | 2500+5 | 2500+800 |
| 5→Max | 5000 | 5000 | 6000+10 | 6000+2000 |

### 3.4 住所等级

住所 = 小院整体等级，决定建筑等级上限和地块上限：

| 住所 Lv | 建筑等级上限 | 地块上限 | 升级消耗 |
|---------|------------|---------|---------|
| 1 | 2 | 4 | — |
| 2 | 3 | 9 | 500金币 |
| 3 | 4 | 16 | 2000金币 |
| 4 | 5 | 25 | 5000金币+3徽章 |
| 5 | Max | 25 | 10000金币+10徽章 |

## 四、铲屎官等级

过关获得经验，升级解锁更多建筑位和宠物饲养数。

| 等级 | 称号 | 所需经验 | 解锁 |
|------|------|---------|------|
| 1 | 实习铲屎官 | 0 | 基础小院 2×2 |
| 2 | 初级铲屎官 | 100 | 可养2只 |
| 3 | 中级铲屎官 | 300 | 地块 3×3 |
| 4 | 高级铲屎官 | 800 | 可养3只 |
| 5 | 资深铲屎官 | 2000 | 地块 4×4 |
| 6 | 金牌铲屎官 | 5000 | 可养4只 |
| 7 | 钻石铲屎官 | 12000 | 地块 5×5 |
| 8 | 传说铲屎官 | 30000 | 可养6只全开 |

**经验来源**：通关一关 +5exp（三星额外+3），每日签到 +10exp。

## 五、游戏循环（通关→结算→返回）

### 5.1 结算面板

```
┌─────────────────────────────────┐
│         🎉 通关！               │
│                                 │
│         ⭐⭐⭐                   │  ← 三星
│                                 │
│   得分：180    步数：12         │
│   🪙金币 +150  🐟小鱼干 +30    │
│   📈经验 +8                     │
│                                 │
│  ┌─────────────────────────┐    │
│  │  📺 看广告奖励翻倍！     │    │  ← 激励视频
│  └─────────────────────────┘    │
│                                 │
│  ┌──────────┐ ┌──────────┐     │
│  │  下一关  │ │ 回主菜单 │     │
│  └──────────┘ └──────────┘     │
└─────────────────────────────────┘
```

### 5.2 流程

```
通关 → 结算面板(自动弹出)
  ├─ 计算得分/步数 → 星级判定
  ├─ 发金币+小鱼干+经验
  ├─ [可选] 看广告 → 货币翻倍
  ├─ "下一关" → 重新加载 GameScene(levelId+1)
  └─ "回主菜单" → SceneManager.LoadScene("MenuScene")
```

### 5.3 星级判定

| 星级 | 条件 |
|------|------|
| ⭐ | 通关 |
| ⭐⭐ | 步数 ≤ 关卡的 minimumMoves × 1.5 |
| ⭐⭐⭐ | 步数 ≤ 关卡的 minimumMoves × 1.2 |

## 九、开发顺序

| 阶段 | 内容 | 依赖 |

| **P0 — 游戏循环** | 通关结算面板 + 下一关/返回 + 星级判定 | 无 |
