```

│    │              │              │               │
│    │              │ 点击正确 ✅   │               │
│    │              │   → +分+特效  │               │
│    │              │              │               │
```

### 1.3 关卡数据结构

```csharp
[CreateAssetMenu(menuName = "FindDiff/LevelData")]
public class LevelData : ScriptableObject
{
    public int levelId;                    // 关卡编号
    public string chapterName;             // 所属章节
    public Sprite imageA;                  // 原图
    public Sprite imageB;                  // 对比图
    public DifferencePoint[] differences;  // 差异点数组
    public int timeLimit;                  // 时限（秒）
    public int difficulty;                 // 难度 1-5
    public int unlockStars;                // 解锁所需星数
}

[System.Serializable]
public class DifferencePoint
{
    public Vector2 center;    // 差异区域中心（归一化坐标 0-1）
    public float radius;      // 点击判定半径
    public string hint;       // 提示文本
}
```

## 游戏2：无厘头大挑战

### 2.1 核心循环

```
┌─────────────────────────────────────────────┐
│                                             │
│  ┌──→ 出题 ──→ 显示题目+选项                │
│  │       ↓                                  │
│  │  ┌─────────────┐                         │
│  │  │  玩家选择    │                         │
│  │  └──┬──────┬───┘                         │
│  │     ↓      ↓                             │
│  │   正确✅  错误❌                          │
│  │     ↓      ↓                             │
│  │  +分      -1生命                          │
│  │  +连击    连击断                          │
│  │     ↓      ↓                             │
│  │  ┌──────────────┐                        │
│  │  │ 还有生命？    │                        │
│  │  │ YES → 下题 ──┘                        │
│  │  │ NO  → 看广告复活？                     │
│  │  │        YES → 回满 💖 + 保留连击 → 下题 │
│  │  │        NO  → 游戏结束                  │
│  │  └──────────────┘                        │
│  │                                          │
│  └─── 结算 ──→ 最高分 ──→ 分享/排行榜       │
│                                             │

```

### 2.3 题库数据结构

```csharp
[System.Serializable]
public class QuestionData
{
    public int id;
    public string category;          // 脑筋急转弯/冷知识/错觉题/文字游戏/热梗
    public string question;           // 题目文本
    public string[] options;          // 4个选项
    public int correctIndex;         // 正确答案索引 (0-3)
    public int difficulty;           // 难度 1-3
    public string explanation;       // 答案解析
    public string illustration;      // 插画文件名（可选）
}

[CreateAssetMenu(menuName = "AbsurdQuiz/QuestionBank")]
public class QuestionBank : ScriptableObject
{
    public QuestionData[] questions;
}
```

### 2.4 出题算法

```
1. 按难度权重随机抽取题目（难度1:60%, 难度2:30%, 难度3:10%）
2. 排除本局已出的题目
3. 随机打乱选项顺序
4. 输出题目 → 等待玩家回答
5. 玩家回答后记录结果 → 调整下一题难度
   - 连续答对3题 → 提升难度权重
   - 答错 → 降低难度权重
```

## 共享系统设计

### 通用模块（两游戏复用）

| 模块 | 说明 |
|------|------|
| GameManager | 游戏生命周期管理 |
| UIManager | 面板打开/关闭/层级 |
| AudioManager | BGM/SFX 播放控制 |
| AdManager | 广告加载/展示/回调 |
| DataStorage | 玩家数据持久化（金币/最高分等） |
| SceneLoader | 场景切换管理 |
| EventCenter | 事件系统（模块解耦） |

### 场景结构

```
Splash.scene (启动加载)
    ├── Main.scene (找茬大师)
    │   ├── Canvas (UGUI)
    │   └── 2D Camera
    └── Quiz.scene (无厘头大挑战)
        ├── Canvas (UGUI)
        └── 2D Camera
```
