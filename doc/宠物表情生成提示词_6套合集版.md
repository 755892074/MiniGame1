## 切图参数

生成合集图后，使用以下命令切图：

```bash
# 通用切图命令（每只宠物）
python tools/cut_animation.py --input cat_expressions.png --output Assets/Art/PetGame/pets/cat/ --frames 7 --target-height 256
python tools/cut_animation.py --input dog_expressions.png --output Assets/Art/PetGame/pets/dog/ --frames 7 --target-height 256
python tools/cut_animation.py --input hamster_expressions.png --output Assets/Art/PetGame/pets/hamster/ --frames 7 --target-height 256
python tools/cut_animation.py --input parrot_expressions.png --output Assets/Art/PetGame/pets/parrot/ --frames 7 --target-height 256
python tools/cut_animation.py --input fish_expressions.png --output Assets/Art/PetGame/pets/fish/ --frames 7 --target-height 256
python tools/cut_animation.py --input rabbit_expressions.png --output Assets/Art/PetGame/pets/rabbit/ --frames 7 --target-height 256
```

**注意**：
- 如果AI没有生成严格的1792×256，切图脚本会自动检测间隙进行智能切分
- 切图后会自动去除白色背景（RGB>245的像素设为透明）
- 每张最终输出为256×256透明PNG
