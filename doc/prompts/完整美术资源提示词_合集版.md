## 切图说明

### 切图脚本位置
```
tools/ai-ui-asset-cutter.py
```

### 使用方法
```bash
# 切横排合集图（等分模式）
python tools/ai-ui-asset-cutter.py --input <合集图路径> --mode force_equal --cols <列数> --output <输出目录>

# 示例：切7个宠物表情
python tools/ai-ui-asset-cutter.py --input cat_expressions.png --mode force_equal --cols 7 --output Assets/Art/PetGame/pets/cat/

# 示例：切15个食物图标
python tools/ai-ui-asset-cutter.py --input foods_sheet.png --mode force_equal --cols 15 --output Assets/Art/PetGame/foods/
```

### 去白底
切图脚本已内置自动去白底功能。如果素材有白底，切完后自动处理为透明背景。
