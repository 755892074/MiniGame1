# 1. 按钮/面板/背景：
#    在 Image2 导出后，用 Photoshop/Photopea 的"切片工具"标记 9-Slice 边距
#    导入团结后，在 Sprite Editor 中设置 Border 值
#
# 2. 道具图标：
#    每张图单独保存透明背景版本（PNG），
#    然后在团结中打 Sprite Atlas 图集
#
# 3. 命名规范：
#    btn_main.png, btn_secondary.png, btn_circle.png
#    panel_dialog.png, panel_toast.png, panel_bottom.png
#    bg_menu.png, bg_levelselect.png, bg_overlay.png
#    item_magnifier.png, item_hourglass.png, item_coin.png, etc.
