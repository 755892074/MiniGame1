软著申请需要什么

软著（计算机软件著作权登记证书）是抖音小游戏上架的前置硬性条件，没有软著连审核都进不去。

软著申请材料

  • 源代码：前30页 + 后30页（共60页），每页不少于50行
  • 软件说明书：设计文档或用户手册（10~15页）
  • 申请表：在中国版权保护中心官网填写
  • 身份证明：个人=身份证复印件；企业=营业执照

关键要求

软著上的软件名称必须与抖音小游戏名称完全一致！
建议名称：铲屎官疯了（全称：铲屎官疯了休闲益智小游戏软件 V1.0）
软著著作权人名称必须与抖音开发者主体名称完全一致。

1 阶段一：软著申请版（当前 → 2周）

目标：整理出可提交软著申请的代码 + 文档

这个阶段不需要游戏完美，只需要"能跑、有完整功能、代码规范"。

需要做的

任务优先级工作量说明
整理源代码文档P01天合并所有.cs为连续文档，前30页+后30页，每页50行
写软件说明书P01~2天10~15页，含功能说明、界面截图、操作流程
确定游戏名称P00.5天名称一旦提交软著不可更改，需同时注册抖音小游戏

2 阶段二：合规版（2~4周）

目标：满足抖音审核的硬性合规要求

这些不做会被审核直接打回。

任务优先级说明
隐私政策弹窗 P0 首次进入弹出隐私协议，用户同意后才进游戏。这是审核第一道关卡
接入抖音 SDK P0 当前是微信小游戏SDK，需接入StarkSDK（抖音小游戏SDK）。团结引擎原生支持
音效系统 P0 至少：倒水声、完成碗声、喂宠物声、通关声、按钮声。审核P1规则明确要求音效正常
防沉迷系统 P0 抖音SDK自带组件，需调用接入。未成年人限时+实名认证
分享功能 P1 分享到抖音聊天/动态，审核会检查是否强制分享（禁止强制）
录屏功能 P1 抖音特色功能，录屏按钮需正常工作，不能打断游戏
自审自查报告 P1 按抖音模板填写，声明无违规内容

3 阶段三：商业化版（4~6周）

目标：接入广告变现，开始赚钱

任务优先级说明
接入穿山甲（CSJ）广告 SDK P0 抖音官方广告聚合，激励视频+插屏。需先在变现中心申请权限
激励视频点位接入 P0 死局+1碗（核心）、提示、撤销、翻倍奖励
插屏广告 P1 关卡间过渡，注意抖音禁止"惯性误触"设计
关卡进度存档 P1 接入PlayerPrefs或抖音云存储，记录已通关数和星级
App Icon + 闪屏 P1 设计游戏图标、Loading界面
抖音广告审核红线（P0违规会直接下架）：
  • 禁止"惯性点击"导致误触广告
  • 禁止隐藏/弱化/延迟广告关闭按钮
  • 禁止默认勾选观看视频广告
  • 禁止把看广告作为关卡解锁的唯一前置条件（可以给奖励但不能卡死）
  • Banner/插屏不能遮挡游戏操作热区

4 阶段四：提审上线（6~8周）

  1. ICP备案：基础信息+版本审核通过后，平台自动申报，7~20个工作日
  2. 小游戏备案：需软著通过 + ICP备案 + 版本审核通过
  3. 版本提审：上传正式版本，等审核（3~5个工作日）
  4. 正式发布：审核通过后上线
